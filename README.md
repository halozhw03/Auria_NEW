# Portfolio Website

A modern portfolio website built with React, TypeScript, and Framer Motion.

## Features

- Split-screen layout (50/50)
- Fixed left panel with brand tagline
- Scrollable right panel for portfolio content
- Smooth animations with Framer Motion
- Inter Light 300 font from Google Fonts
- Primary color: #F5F0E7

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```
